jQuery ScrollEasy

A lightweight and highly customizable plugin for perfect smooth scrolling.

Why choose ScrollEasy?

This plugin transforms default, jumpy HTML anchors into elegant, fluid animations. It is ideal for One-Page websites, landing pages, and technical documentation.

  • Lightweight and fast execution
  • Smart offset support (perfect for fixed navigation menus like this one)
  • Custom callbacks fired upon animation completion

How the Offset works

Have you noticed that when you click the links in the top menu, the section titles are never hidden behind the dark navigation bar? This happens seamlessly thanks to the offset: 60 parameter set during plugin initialization.

Try scrolling up and down using the menu above to see it in action!
